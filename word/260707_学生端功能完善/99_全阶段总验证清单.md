## 2. 学生端端到端用户旅程（浏览器，一个学生账号）

前置：教师已发布含 1 客观题 + 1 主观题的任务到该学生所在班级。

1. 学生登录 → 首页看到任务（**F1**：即使后入班也看得到）。
2. 点任务 → 关卡 → 答客观题（错）→ ❌ 反馈，提示"进入引导模式"（**F3**）。
3. 答主观题 → ⏳"已提交，待批阅"，无引导入口、不进错题本（**F3**）。
4. 进引导（C 模式）→ 出现真实 LLM 评价 + 步进问题，无"注意观察图形"（**F7**）。
5. 错题本 → 看到该客观错题 → 同类题 → 题目带题干 → 开始练习（**F4**）。
6. 同类题答对 3 次 → 错题状态推进 mastered、掌握度更新（**F4**）。
7. 成长页 → 正确率不被主观题拉低（**F3**）。
8. 知识图谱页 → 真实知识点掌握度列表（**F9**）。
9. 错题本/任务 → 导出 PDF → 下载 `.pdf`，中文正常（**F8**）。
