## 3. 变式题推荐

### 3.1 知识点匹配验证

**验证方法：** API 请求或 Django Shell

```bash
# 方式 1: API 请求
curl http://localhost:8000/api/v1/student/wrong-book/<wrongbook_item_id>/variants \
  -H "Authorization: Bearer <student_token>"

# 方式 2: 直接调用函数
python manage.py shell -c "
from apps.wrongbook.services import find_variant_questions
# 找一个有 knowledge_points 的题目
from apps.parser.models import ExamQuestion
q = ExamQuestion.objects.exclude(knowledge_points__isnull=True).exclude(knowledge_points=[]).first()
if q:
    print('Original question:', q.id, 'knowledge_points:', q.knowledge_points)
    variants = find_variant_questions(q.id, limit=3)
    print('Variants found:', len(variants))
    for v in variants:

else:
    print('No question with knowledge_points found')
"
```

**预期结果：**
- [ ] 有知识点的题目，优先推荐同知识点的变式题

- [ ] 无知识点的题目，按原逻辑（学科+难度）匹配
- [ ] 返回结果包含 `id`, `stem`, `options`, `images` 等字段
