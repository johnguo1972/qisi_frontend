## 修改顺序建议

1. 先修改 `config/settings.py`（后端配置）
2. 再修改 `uniapp` 相关文件（manifest.json, request.ts, .env.production, vite.config.ts）
3. 然后修改项目根目录 `.env`
4. 最后修改 `start.bat`
5. 全部修改完成后，重新构建前端 H5：`cd uniapp && npm run build:h5`
6. 确认本地开发环境正常运行后再部署到服务器
